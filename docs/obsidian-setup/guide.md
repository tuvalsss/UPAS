# How to Open This Vault in Obsidian

1. Install Obsidian from https://obsidian.md
2. Open Obsidian → **"Open folder as vault"** → select the `docs/` folder
3. Enable core plugins: **Graph view**, **Backlinks**, **Outgoing links**, **Search**
4. Open Graph View (`Ctrl+G`) to see the full system map
5. Start from [[HOME]]

## Recommended Community Plugins

- **Dataview** — query notes by tag/type as live tables
- **Graph Analysis** — enhanced graph with clustering
- **Tag Wrangler** — manage tags across the vault

## Graph View Tips

- Filter by tag `agent` → agents only (blue)
- Filter by tag `strategy` → full strategy graph (orange)
- Filter by tag `tool` → tool layer (cyan)
- Filter by tag `index` → all index nodes (red)
- Use **depth** slider to expand/collapse neighborhoods
- Color groups are pre-configured in `.obsidian/graph.json`
